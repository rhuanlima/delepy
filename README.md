# DelePy
É um app em python desenvolvido para construir apresentações em HTML de fácil acesso a leitores de tela como JAWS, ORCA ou mesmo o Voice Over

## Como funciona?
O Pacote recebe um arquivo com os parametros em markdow e os converte para um pack em html com o material em apresentação

## Templates
Na pasta Templates temos o html usando JINJA para composição de seus temas de apresentações, por padrão temos um template default e para mais infos foi criado um readme na pasta para auxiliar na utilização
